Overview

doVOUCHER — service for creating multiple vouchers in cryptocurrency and NFT, which can be given to users via a link or print QR-code for marketing solutions.

Task

To develop a modern and convenient web service where the user could create a voucher by recharging an account or prefixing NFT, via phone or computer, as well as to provide possible to see in a private office statistics on created and gifted vouchers.

It has been done:

  • Web service
  • 16 unique interior pages
  • Landing page
  • User's personal cabinet with statistics on vouchers
  • All adapted for the phone
